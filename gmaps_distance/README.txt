Command line tool built in python
params: source, destination
output: distance between the two, using google maps api

1. Open 'gmapdist' in an editor, edit in your Google Distance Vector API key.
2. Set executable permissions on the script if they are not set already.
3. sudo cp gmapdist /usr/bin
4. (command line) gmapdist "src" "dest"

You can get a key for free here:
https://developers.google.com/maps/documentation/distance-matrix/intro
